F1 SKU Master (P1 → D1)

จัดการตาราง sku_master_data — รายการ SKU ที่คาดว่าจะนับในแต่ละคลัง ใช้คำนวณ KPI 「ยังไม่ได้นับ」「นับแล้วใน Master %」

1.1 เครื่องมือบนหน้า

  • เลือกคลัง — ตึกกันตนา / หน้าไลฟ์ / คลังอะไหล่ (Master แยกต่อคลัง)
  • อัปโหลด Excel — นำเข้ารายการ SKU (คอลัมน์หลักคือรหัส SKU)
  • Preview ก่อนยืนยัน — ตรวจจำนวนแถว / แถวซ้ำ / คอลัมน์ไม่ครบ
  • บันทึก (Upsert) — อัปเดตแถวเดิมหรือเพิ่มใหม่ตาม SKU+คลัง
  • ดู/ลบรายการในตาราง — จัดการ Master หลังนำเข้า

1.2 ขั้นตอนนำเข้า SKU

  1. เลือกคลัง ที่ต้องการ
  2. เลือกไฟล์ .xlsx ที่มีคอลัมน์ SKU
  3. ตรวจ preview — แก้ไฟล์ถ้ามีแถว error
  4. กดยืนยันนำเข้า — รอจนเสร็จ

1.3 สินค้าใหม่ (นอก Master)

SKU ที่นับแล้วแต่ไม่อยู่ใน Master จะไม่นับใน % ความคืบหน้า แต่แสดงที่หน้า นับสต็อก ในแบนเนอร์「สินค้าใหม่」— Export ไปเพิ่มใน Master ภายหลัง

ควรอัปเดต Master ก่อนเริ่มรอบนับ ของเดือน เพื่อให้ KPI และรายการ「ยังไม่ได้นับ」ตรงกับของจริง

F2 รอบนับ + ไฟล์ Excel Book (P2 → D2, D3)

ศูนย์กลางรอบนับ — สร้างรอบ อัปโหลดยอด Excel (Book) ผูกผลนับ ตั้งรอบ active

6.1 สร้างรอบใหม่

  • คลังเดียว — ตึกกันตนา / หน้าไลฟ์ / คลังอะไหล่
  • หลายคลัง — เลือก checkbox หลายคลัง (เก็บ warehouse แบบ คลัง1|คลัง2)
  • คลังทั้งหมด — Book ไฟล์เดียวรวมทุกคลัง · ต้องกำหนดวันเริ่ม-สิ้นสุด
  • ปี + เดือน · ช่วงวันที่นับ (ว่าง = ทั้งเดือน)
  • ชื่อรอบ (ไม่บังคับ)

6.2 อัปโหลดไฟล์ Excel (Book)

  1. เลือกรอบในตาราง
  2. เลือกไฟล์ Excel (SKU + จำนวน)
  3. ดู preview — แถว ไม่ผ่าน (ไม่มี SKU / จำนวนไม่ถูก) แสดงตารางแยก
  4. กดอัปโหลด — เข้า book_stock_lines

6.3 ผูกผลนับ + ใช้งานรอบ

ผูกผลนับกับรอบนี้ — อัปเดต cycle_id ตามช่วงวัน (ไม่แก้ qty) · ใช้งาน — บันทึก active ใน localStorage สำหรับ insert ใหม่

กด「โหลดเดือนที่มีข้อมูลนับ」เพื่อเติม dropdown เดือนจากข้อมูลจริงใน DB

F6 Match ยอด (P5 → D5, D6)

เทียบ ไฟล์ Excel (ยอดก่อนนับ) กับ ผลนับรวม ต่อ SKU ในรอบที่เลือก

7.1 เครื่องมือ

  • เลือกคลัง + รอบ
  • คำนวณ Match / Refresh — เรียก RPC อัปเดต reconciliation_lines
  • KPI — ตรง · ขาด · เกิน · ยังไม่ได้นับ · นับเจอแต่ไม่พบใน Excel
  • ตัวกรองสถานะ ในตาราง
  • ปรับยอด (draft) — บันทึกที่ stock_adjustments ฝั่ง Excel
  • Export Excel

7.2 สถานะและคอลัมน์

สถานะความหมาย
ตรงผลนับ = Excel ใช้เทียบ (หลังปรับ)
ขาดนับ < Excel
เกินนับ > Excel
ยังไม่ได้นับมีใน Excel ไม่มีผลนับ
นับเจอ แต่ไม่พบSKUในExcelมีผลนับ ไม่มีใน Excel

7.3 สูตร

Excel ใช้เทียบ = SUM(book_qty) + SUM(adjustments applied)
ผลนับ = SUM(inventory_counts.counted_qty)  ← อ่านอย่างเดียว
variance = ผลนับ − Excel ใช้เทียบ
ห้ามแก้ counted_qty ใน inventory_counts เพื่อให้ Match ตรง

10.1 โหมดอ่านอย่างเดียว
